# Battleship Game in C

A console-based Battleship game developed in ANSI C. The game features randomly placed ships, input validation, file logging, and ship tracking.

## Features

- 10×10 game board
- Random ship placement
- Four ships with different lengths
- Hit and miss tracking
- Ship sunk detection
- Input validation
- File logging
- Restart option

## Technologies

- ANSI C
- Standard C Library
- File I/O

## Compile

```bash
gcc battleship.c -o battleship
```

## Run

```bash
./battleship
```
